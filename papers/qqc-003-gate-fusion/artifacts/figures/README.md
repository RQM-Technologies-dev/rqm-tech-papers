# Figures — qqc-003-gate-fusion

This directory contains the figures used in
**Quaternionic Gate Fusion on S^3: Geometry-Native Optimization for Single-Qubit Circuit Segments**.

## Included figures

| File | Description |
|------|-------------|
| `fig-fusion-pipeline.svg` | Pipeline from named single-qubit segment syntax to quaternion IR, fused canonical representative, and reconstructed output |
| `fig-canonical-hemisphere.svg` | Canonical hemisphere and equatorial tie-break used during sign-canonicalization |
| `fig-segment-boundaries.svg` | Example of segment extraction inside a larger circuit with entangling boundaries |
| `fig-before-after.svg` | Before/after optimization view for small single-qubit segments |

All figures are publication-oriented SVG schematics intended to explain the
algorithm and evaluation setup. They do not encode benchmark data beyond the
worked examples already reported in the paper and benchmark JSON artifact.
