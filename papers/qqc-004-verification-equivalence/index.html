qqc-004-verification-equivalence

Package landing page for Verification, Canonicalization, and Equivalence in Quaternionic Quantum Compilation. This paper defines the trust layer for quaternionic compilation by specifying canonical equivalence checks, preserved invariants, matrix-interoperable diagnostics, and reproducible validation procedures.

Read the HTML paper Open the PDF

Package overview

The paper depends on papers 1--3 and contributes a verification layer for canonical quaternionic compilation, including a shipped reference verifier and measured positive/negative comparison cases.
