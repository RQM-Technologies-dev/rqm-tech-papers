# Figures — qqc-004-verification-equivalence

This directory contains the publication-oriented SVG figures used in
**Verification, Canonicalization, and Equivalence in Quaternionic Quantum Compilation**.

## Included figures

| File | Description |
|------|-------------|
| `fig-verification-pipeline.svg` | End-to-end trust pipeline from optimized segment to canonical quaternion, matrix diagnostics, and verification decision |
| `fig-canonical-hemisphere.svg` | Canonical hemisphere and equatorial tie-break used to make quaternion representatives reproducible |
| `fig-matrix-interoperability.svg` | Relationship between quaternion representatives, the fixed `Phi` map, and matrix-based diagnostics such as trace overlap and process fidelity |
| `fig-positive-negative-cases.svg` | Positive and negative verification examples showing expected pass/fail behavior |

All figures are vector schematics intended to clarify the verification and trust
story of the paper rather than to encode hardware or experimental data.
