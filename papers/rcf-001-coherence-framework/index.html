Package landing page for RQM Coherence Framework v0.1: Spectral-Variational Coherence on S^3 x R. This draft formalizes coherence as a residual-to-eigenform operator property and defines a shared metric layer for RQM Technologies products and internal technical workflows.

Package overview

This package introduces the first numbered paper in the RQM Coherence Framework (RCF) series. The source-of-truth manuscript is authored in main.tex, and the package includes synchronized machine-readable companions for claims, notation, glossary, and metadata indexing.
