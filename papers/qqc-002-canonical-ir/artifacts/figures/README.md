# Figures — qqc-002-canonical-ir

This directory contains figures for
**A Canonical Quaternionic Intermediate Representation for Single-Qubit Quantum Circuits**.

## Included figures

| File | Description |
|------|-------------|
| `fig-ir-pipeline.svg` | Flow from named single-qubit gate syntax to quaternion tuples and deterministic canonical `u1q` representatives |
| `fig-hemisphere-canonicalization.svg` | Canonical hemisphere and equatorial tie-break for the sign ambiguity `q ~ -q` |
| `fig-segment-extraction.svg` | Example of extracting a single-qubit segment between entangling-gate boundaries |

All figures are vector schematics intended to clarify the IR contract and canonicalization rules. They are expository figures, not performance plots.
