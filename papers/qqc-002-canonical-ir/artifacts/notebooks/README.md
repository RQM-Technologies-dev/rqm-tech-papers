# Notebooks — qqc-002-canonical-ir

This paper package does not currently include executable notebooks.

The present draft defines a canonical single-qubit intermediate representation and
its deterministic normalization rules. Future notebooks, if added, should remain
supporting material rather than primary evidence. Suitable notebook topics include:

- checking quaternion multiplication and re-canonicalization on small gate lists,
- plotting the canonical hemisphere and equatorial tie-break convention, and
- validating named-gate translation tables against symbolic or numerical matrix forms.

Any future notebook should state clearly whether it is pedagogical, regression
testing support, or implementation prototyping for a downstream compiler repository.
