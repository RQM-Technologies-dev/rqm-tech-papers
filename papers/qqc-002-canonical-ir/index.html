Paper package

qqc-002-canonical-ir

Package landing page for A Canonical Quaternionic Intermediate Representation for Single-Qubit Quantum Circuits. This paper defines a deterministic quaternionic IR layer for single-qubit gate segments modulo global phase.

Read the HTML paper Open the PDF

Package overview

The paper depends on qqc-001-foundations for the fixed quaternion--SU(2) convention and contributes a canonical u1q(w,x,y,z) representation layer for compiler pipelines.
