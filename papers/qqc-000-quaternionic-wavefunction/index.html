Package landing page for The Quaternionic Wavefunction: Spin-1/2 Particles, S^3, and SU(2). This package is intended as a conceptual prequel to the rest of the quaternionic quantum computing series.

Read the HTML paper Open the PDF

Package overview

This prequel introduces the quaternionic wavefunction as an expository bridge from a two-component spinor in C^2 to the geometry of S^3 and the transformation group SU(2), with an emphasis on spin-1/2 particles and the 4 pi property.
